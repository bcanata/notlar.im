[mmm.page](https://build.mmm.page) is a simple tool for designing and publishing a personal website. You can quickly build a personal site by drag-and-dropping every element onto the page and positioning it.

See [https://mmm.page/bc.main](https://mmm.page/bc.main) for the personal page I built for myself as an example.
