Aslında daha çok, sağda solda gördüğüm, kendi işime yarayacak GitHub repoları. Yıldızladıklarımı buraya atıp, açıklamasını yazıyorum.

- [YS-L/csvlens](https://github.com/YS-L/csvlens) — Komut satırı CSV okuyucu.
- [wagoodman/dive](https://github.com/wagoodman/dive) — Docker imajlarının içerisini katman katman incelemeye yarıyor.
- [pocketbase/pocketbase](https://github.com/pocketbase/pocketbase) ([site](https://pocketbase.io/)) — Go ile yazılmış efsane bir paket. Tek tıkla hazır bir veritabanı, dosya saklama, kullanıcı girişi yapma vs. için admin panelli bir backend sunuyor. JSON API ile istediğinizi bağlayıp kullanabilirsiniz.
- [bcanata/awesome-stars](https://github.com/bcanata/awesome-stars/) — "awesome-stars" şeklinde bir repo varmış; yıldızladığım repoları oluşturup liste hâlinde paylaşıyor.
